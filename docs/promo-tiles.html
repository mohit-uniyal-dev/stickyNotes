Stick it - web notes — Chrome Web Store assets

Click a “Download PNG” button to save that asset at its exact required size — the marquee promo tile (1400×560) and the small promo tile (440×280). Everything is drawn as SVG and exported through a canvas, so each PNG is pixel-perfect regardless of your screen. If a download ever looks off, right-click the graphic → Inspect → right-click the <svg> node → “Capture node screenshot” as a fallback.
