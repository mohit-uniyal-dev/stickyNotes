Welcome to Stick it - web notes

Add sticky notes to any web page and pick up right where you left off. Everything is saved on your device — no account, no servers.

Getting started

Add a note

Open Stick it from the toolbar on any website and click Add Note — a sticky note appears right on the page, ready to type in.

It stays put

Pin a note and it comes back automatically the next time you visit that page. Notes are tied to the exact page URL.

Make it yours

Drag, resize, recolor, and minimize each note. Minimized notes tuck into a tidy tray in the corner.

Good to know

Pin the extension

Click the puzzle-piece icon in the toolbar and pin Stick it so it's always one click away.

All your notes in one place

Open the All Notes page to search, edit, and manage every note, grouped by website.

Private by design

Your notes never leave this computer. No accounts, no tracking, no servers.

Tip: open Stick it on any web page and click Add Note to get started.
